# GyraNō 🧠

AI-powered browser extension that summarizes YouTube videos instantly.

## 🚀 Features
- Instant AI summaries
- Clean dark-mode UI
- Lightweight and fast

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Node.js (if backend)
- OpenAI API (if used)

## 📦 Installation

1. Clone the repo
2. Open Chrome → Extensions
3. Enable Developer Mode
4. Click "Load Unpacked"
5. Select project folder
